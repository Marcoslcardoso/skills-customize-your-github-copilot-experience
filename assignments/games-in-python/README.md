
# 📘 Assignment: Hangman Game

## 🎯 Objective

Build an interactive Hangman game in Python to practice string manipulation, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Game Loop and Word Progress

#### Description

Develop the core hangman logic to choose a secret word, receive letter guesses, and show the current progress with hidden letters.

#### Requirements
Completed program should:

- Select a random word from a predefined list
- Display the secret word with underscores for unguessed letters
- Accept letter guesses from the user
- Update and display the current progress after each guess

### 🛠️ Attempts and Win/Loss Conditions

#### Description

Implement attempt tracking and game end conditions so the player can win or lose correctly.

#### Requirements
Completed program should:

- Track incorrect attempts remaining
- Decrease attempts when the user guesses a wrong letter
- End the game when the word is fully guessed
- End the game when attempts run out
- Show a victory message when the word is guessed and a defeat message when the player loses
